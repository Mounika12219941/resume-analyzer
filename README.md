# 🚀 AI Resume Analyzer

A simple AI-based Resume Analyzer built using Flask.

## 🔥 Features
- Upload resume
- Word count analysis
- Skill detection (Python, Java, etc.)
- Includes ATS scoring system and resume improvement suggestions

## 🛠 Tech Stack
- Python
- Flask
- HTML

## ▶️ How to Run
pip install flask  
python app.py
